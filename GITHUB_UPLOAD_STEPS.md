# GitHub Upload Steps

## Website upload method

1. Create a new empty GitHub repository.
2. Do not select `Add README`, `.gitignore`, or license because this project already includes GitHub-ready files.
3. Extract the ZIP file on your laptop.
4. Open the extracted folder.
5. Select all files and folders inside the project folder, including hidden files like `.gitignore` and `.github`.
6. Drag and drop them into the GitHub upload page.
7. Commit with this message:

```text
Initial commit - Docker showcase portfolio
```

## Recommended repository name

```text
docker-showcase-portfolio
```

## Recommended GitHub description

```text
Docker portfolio with Django/Nginx, Django/React/PostgreSQL, and MERN/MongoDB multi-container examples using Docker Compose.
```

## Important

Do not upload dependency folders such as `node_modules`, Python virtual environments, local logs, or database data folders.
